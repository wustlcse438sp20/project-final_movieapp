Our Movie App is designed for people to explore good movies, search movies and create personalized movie lists.

Our app has several key features:

1) within the movie detail page, a section of similar movie recommendations are provided to the user. So the user can explore movies 
that are similar to his/her liked movies.

2) within the movie detail page, a section of review from movie critic is presented. So user can look at the review of this movie from 
a movie expert.

3) we offer users the ability to create his/her personalized movie list, so he/she can add movie to the list when he/she finds a good movie

4) we also offer the function to rate a movie, and the rated movie will be ranked based on their rating under the trending tab.

5) user has to register or login before he/she can use the app.

6) user can search a genre, or a specific movie title in the search box.
